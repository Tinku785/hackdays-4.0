**Hackdays 4.0** is an immersive, high-energy landing page for the annual technical fest of the **Department of Information Technology, Gauhati University**. 

Designed and developed with a signature **90s Marvel Comic Book** aesthetic, the site blends retro neobrutalism with futuristic **Stark Industries** technology.

Developed by **Tinku Moni Kaushik** in collaboration with **Agentic AI**.

---

## 🚀 Key Features

### 💻 Stark OS Terminal
An interactive command-line interface directly from Tony Stark's lab. 
- Try typing `help`, `jarvis`, `matrix`, or `protocol --shield`.
- Fully responsive and accessible on mobile.

### 🦹 Hero & Villain Modes
Toggle the entire site's reality! 
- **Hero Mode**: Vibrant reds, yellows, and classic Spider-Man vibes.
- **Villain Mode**: Neon greens, deep blacks, and a sinister Venom-infused theme.

### 📱 Mobile-First "App-Like" UX
Optimized for the modern "True Believer."
- **Glassmorphism Bottom Nav**: A persistent, premium navigation bar with steel-shined animated icons.
- **Smart Formatting**: Auto-scaling typography and layout slants designed specifically for mobile viewports.
- **Touch-Optimized**: Custom cursors are disabled for touch, and modals have expanded touch targets.

### ⚡ Interactive Elements
- **Spiderman/Venom Scroll**: A dynamic character that hangs from the top as you explore the site.
- **Parallax Action Bubbles**: "POW!", "ZAP!", and "BOOM!" elements that react as you scroll.
- **Interactive Dot Grid**: A high-performance background grid that responds to mouse and touch interactions.
- **Comic Dossier Modals**: Mission intel presented as top-secret classified documents.

---

## 🛠️ Technology Stack

- **Framework**: React.js with Vite
- **Animations**: GSAP, Framer Motion
- **Icons**: Lucide React
- **3D Effects**: Three.js & Custom Canvas logic
- **Styling**: Vanilla CSS (Modern Neobrutalism)

## 📦 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v16.x or higher)
- [npm](https://www.npmjs.com/)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/your-repo/hackdays4.0.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
